Detailed Description

template<typename T>
struct BinarySearchTree< T >::Node

It's the basic building block of BinarySearchTree.

Nodes store informations in BST. They are internally connected, via pointers to left, right and parent. Left child of the node is smaller than it's parent and right child is greater than the given node.

This implementation represents Nodes as a struct, because I didn't see the purpose in creating the implementation of additional functions inside Node that would be present in class representation.
